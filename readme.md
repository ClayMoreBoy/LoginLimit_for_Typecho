### LoginLimit ##
---
#### 起步
`var/Widget/User.php`中约117行，将这三行注释，因为这边有执行插件就返回插件的返回结果，而插件并没有验证登录逻辑。

![微信截图_20151207152313.png][1]


#### 使用方法：
第一步：下载本插件，放在 `usr/plugins/` 目录中；  
第二步：激活插件；  
第三步：设置30分钟允许登录失败次数；  
第四步：完成。

#### 管理：
管理后台-管理-登录日志：可查看别人尝试登录系统的记录。
